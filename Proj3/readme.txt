Como jogar:
1º Abrir o SICStus, fazer uma consulta ao ficheiro server.pl
2º Abrir o mongoose
3º No browser escolhe-se agora as configurações de jogo
4º Carregar no botão "Play Game" para jogar
5º O jogo obedece às regras do ModX (https://boardgamegeek.com/boardgame/131387/mod-x)

P.S.: O botão "Quit" serve para "desligar" o SICStus
